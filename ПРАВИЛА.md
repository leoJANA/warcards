# Война за Хакон — правила боя (v2)

Карточный варгейм для режима глобальной войны в ДнД. Игрок ведёт армию против ГМ.
Стиль вдохновлён Legends of Runeterra, механика своя.

## Цель
У каждой стороны есть **Командир (Штаб)** с **20 ❤**. Победа — снизить здоровье Командира противника до 0.

## Ресурс
- **Найм (мана)** растёт каждый раунд: раунд N даёт `min(N, 10)` маны, в начале раунда восполняется.

## Статы отряда
- **Стоимость найма** — сколько маны нужно, чтобы вывести отряд.
- **⚔ Урон** — сколько урона наносит в атаке.
- **🛡 Защита (броня)** — снижает каждый получаемый удар.
- **❤ ХП (прочность)** — запас здоровья. При 0 ❤ отряд гибнет.

**Урон по ХП = Урон атакующего − Защита цели** (минимум 0).
**ХП не восстанавливается между раундами** — раны копятся.

## Структура раунда
Жетон атаки чередуется между сторонами каждый раунд.

1. **Найм.** Сначала атакующий, затем обороняющийся выводят отряды и жмут «Готово».
   Каждая выведенная карта крупно показывается обоим игрокам со статами.
2. **Атака.** **Фазы блока нет.** Атакующий сам выбирает цель каждому своему отряду:
   - кликает свой отряд → кликает вражеский отряд **или** баннер вражеского Командира;
   - по Командиру можно бить, только если у врага **не осталось отрядов** (или у атакующего «Дальний бой»);
   - бой однонаправленный: цель получает урон, но не контратакует.
3. Новый раунд, жетон атаки переходит другой стороне.

## Ключевые слова
- ⚡ **Стремительный** — может атаковать в тот же раунд, когда выведен.
- 💥 **Натиск** — излишек урона сверх ХП цели проходит Командиру.
- 🛡️ **Барьер** — поглощает первую атаку по нему за раунд.
- 🏹 **Дальний бой** — может бить Командира напрямую, даже если у врага есть отряды.
- 🪨 **Стойкость** — получает на 1 меньше урона от каждой атаки (сверх защиты).
- 🔥 **Ярость** — выжив после полученного урона, навсегда получает +1 ⚔.

## Режимы
- **🧪 Тест: ∞ маны** — у обеих сторон бесконечная мана (для отладки и обкатки карт).
- **Глаз ГМ** — показывает обе руки (для ведущего).

## Дальше (план)
- Сетевой режим: игроки заходят по ссылке и играют против ГМ.
- Редактор карт: ГМ создаёт свои отряды.
- Привязка к кампании: исходы войны меняют ситуацию в мире ДнД.
